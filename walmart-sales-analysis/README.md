# Walmart Sales Analysis

## Objective
Analyze sales trends, seasonal patterns and holiday impact using historical Walmart sales data.

## Tools Used
Python (pandas, matplotlib)
SQL
Power BI

## Work Done
- Cleaned and prepared raw sales data
- Analyzed monthly and holiday sales patterns
- Compared holiday vs non-holiday performance
- Created dashboard for business insights

## Key Insight
Holiday periods contributed lower overall sales compared to regular periods.
